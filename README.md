# Purpose
  
PyTorch development container, backend TensorFlow.

### Feature

- PyTorch 0.3.0, Cuda 9
- Torchvision 0.2.0
- TensorboardX
- tqdm
- Scikit Image

### Dependency

- [TensorFlow](https://github.com/rainbean/Container_TensorFlow)

### Usage

Docker pull command
```
$ docker pull rainbean/pytorch
```
